# HTML Batch 01

This repository contains complete HTML5 training materials, examples, and practice projects created during the HTML learning journey.

## Topics Covered
- HTML Basics (Headings, Paragraphs, Links, Images)
- Lists and Tables (Basic & Advanced)
- Forms and Input Types
- Multimedia (Audio, Video, Iframe)
- Semantic HTML Tags
- Meta Tags and Page Structure
- Fieldset & Legend
- HTML Entities
- Accessibility Basics (alt, labels, aria attributes)

## Purpose
To gain hands-on experience in HTML5 and build strong fundamentals of web page structure and development.

## Status
Completed HTML fundamentals with practical implementation.
